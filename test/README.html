Take-home javascript exercise

In this folder you will find a simple html page and accompanying js/css files:

  • index.html
  • api.min.js
  • udsTable.css
  • yourJsFile.js

api.min.js contains some methods for you to use to simulate retrieving data, as well as a helper method for building a table from a datasource. You will use yourJsFile.js to complete some steps for fetching data, processing it, and presenting it. You will find a comment and a couple lines there already, providing documentation on the data api methods and the table helper method.

  1. Get names and counts from the data api, and combine them into one data source.
  2. Add another field to the data: percent of total. It should equal count divided by the total of all counts. For example, if the total of all counts is 200, a data item might have { count: 5, percentOfTotal: .025 }
  3. Using the table helper method, create a table from this data showing name, count, and the percent of total. Format the last as a percentage, with 2 decimal places (e.g. "2.50%").

Try to minimize the complexity of the algorithms you use for each step.

When you are finished, zip up all the files and attach them in a reply. I should be able to simply open index.html and see your solution in action. Good luck!

Extra credit items:

  • Show an error message if the data retrieval fails.
  • Show an indication on the page while the data is still loading.
  • Sort the list alphabetically (how efficiently can you do this? Hint: memory is cheap, time is not)
